# 🛒 DigiTools

A modern digital tools marketplace to browse and purchase premium productivity and AI tools.


## 🌐 Live Site
🔗 [https://demo-assignment-gamma.vercel.app/](https://demo-assignment-gamma.vercel.app/)


## 🛠️ Technologies Used
- React.js
- Tailwind CSS & DaisyUI
- React-Toastify
- JSON

## ✨ Features
1. 🛍️ Add to cart and proceed to checkout with toast notifications
2. 🔄 Real-time cart count updates in the navbar
3. 📱 Fully responsive design across all devices
